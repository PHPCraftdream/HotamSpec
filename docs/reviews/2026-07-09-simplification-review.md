## Сходящиеся находки (подтверждены 2+ агентами независимо)

1. **Attention-шум**: 38 из 40 сигналов — идентичный P0 REFLECTION («мигрируй REJECTED на structural replaces-edge»), помеченный «Advisory; never a gate» — оксюморон для P0. Реальный открытый вопрос тонет. Фикс: batch-миграция закрывает всё разом.
2. **Две параллельные системы делегаций**: `record_delegation.py`/jsonl (устаревшая, R-trust-anchor-delegation-explicit-only) vs `delegate.py`/DG-файлы (принятая, R-delegation-is-a-file). Первая не выведена из эксплуатации.
3. **Entity/Process/Goal/Operator-типы почти не используются** (Goal — 1 инстанс, Process — 1, EntityType — 0 в hotam-spec-self) — балласт в горячем пути тестов.
4. **`gen_spec.py` (4786 строк) сцеплен архитектурно**: контракт — байты markdown-вывода, а не структурная модель, поэтому безопасный распил невозможен (подтверждено и волной 6.2b этой же сессии).
5. **Портативность не имеет consumer-документации**: README не знает о `pip install`/29 CLI-командах.
